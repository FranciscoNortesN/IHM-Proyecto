# Sistema de Problemas

## Introducción a los problemas

El sistema de problemas de NavTrainer está diseñado para ayudarte a practicar y mejorar tus habilidades de navegación. Los problemas cubren diferentes aspectos de la navegación náutica, desde cálculos básicos hasta situaciones complejas de navegación.

## Tipos de problemas

### Problemas de rumbo

Calcula rumbos verdaderos, magnéticos y de aguja:

- Conversión entre rumbos
- Corrección de declinación magnética
- Corrección de desvío
- Rumbo efectivo considerando viento y corriente

### Problemas de distancia

Mide y calcula distancias:

- Distancia entre dos posiciones
- Distancia navegada en un tiempo dado
- Cálculo de ETA (tiempo estimado de llegada)
- Consumo de combustible según distancia

### Problemas de posición

Determina tu posición en el mar:

- Posición por demarcaciones
- Círculos de posición
- Posición por enfilaciones
- Triangulación usando puntos conocidos

### Problemas de navegación costera

Situaciones de navegación cerca de costa:

- Paso de puntos notables
- Navegación por canales
- Fondeo seguro
- Aproximación a puerto

### Problemas de navegación astronómica

Navegación usando cuerpos celestes:

- Altura del sol al mediodía
- Líneas de posición astronómicas
- Cálculo de latitud por Polar
- Determinación de la hora

## Selección de problemas

### Modo selección manual

Elige problemas específicos:

1. Ve a la sección "Problemas"
2. Explora las categorías disponibles
3. Selecciona el problema que quieras resolver
4. Haz clic en "Comenzar"

### Modo aleatorio

Genera problemas aleatorios para práctica variada:

1. Haz clic en "Problema aleatorio"
2. Opcionalmente filtra por categoría o dificultad
3. El sistema generará un problema con datos aleatorios
4. Resuelve y envía tu respuesta

**Notas de la ventana de ejercicio**

- Las ventanas de problema se muestran como ventanas separadas con título `Ejercicio N` o `Ejercicio N (Aleatorio)` según procedencia.
- Puedes minimizar un problema mientras mantienes tu trabajo: las sesiones de problema se pueden abrir y cerrar sin perder el estado si no las cierras permanentemente.
- El área de la carta dentro del problema comparte las herramientas estándar (Regla, Compás, Transportador) para que realices la resolución directamente sobre la carta proporcionada.

## Resolver problemas

### Interfaz de resolución

La pantalla de problema incluye:

- **Enunciado**: Descripción detallada del problema
- **Carta**: Área de la carta náutica relevante
- **Herramientas**: Acceso a regla, transportador, compás
- **Campos de respuesta**: Donde introduces tus respuestas

### Proceso de resolución

1. **Lee cuidadamente** el enunciado del problema
2. **Identifica** qué datos te dan y qué necesitas calcular
3. **Usa las herramientas** disponibles en la carta
4. **Calcula** la respuesta usando los métodos apropiados
5. **Introduce** tu respuesta en los campos correspondientes
6. **Verifica** tus cálculos antes de enviar
7. **Envía** tu respuesta cuando estés seguro

### Uso de herramientas

Durante la resolución puedes usar:

- **Todas las herramientas de mapa**: Regla, transportador, compás
- **Calculadora**: Para cálculos numéricos
- **Tablas**: Tablas de declinación, desvío, etc.
- **Notas**: Espacio para tus anotaciones

## Retroalimentación

### Respuesta correcta

Cuando aciertas:

- Se muestra un mensaje de felicitación
- Ves tu puntuación obtenida
- Se destaca la solución en la carta
- Puedes ver la explicación detallada

### Respuesta incorrecta

Cuando fallas:

- Se indica dónde está el error
- Puedes intentarlo de nuevo

## Historial de problemas

### Registro de intentos

El sistema guarda:

- Todos los problemas que has intentado
- Tus respuestas y puntuaciones
- Fecha y hora de cada intento

### Revisar problemas anteriores

Puedes volver a problemas anteriores:

1. Ve a "Historial de problemas"
2. Selecciona el problema que quieres revisar
3. Ve tu solución anterior
4. Compárala con la solución correcta
5. Intenta resolverlo de nuevo si quieres mejorar

## Práctica efectiva

### Consejos para mejorar

- **Practica regularmente**: Mejor 15 minutos al día que 2 horas una vez a la semana
- **Varía los tipos**: No te quedes solo con problemas que dominas
- **Revisa errores**: Aprende de tus fallos
- **Usa modo aleatorio**: Para enfrentarte a situaciones inesperadas
- **Trabaja sin ayudas**: Cuando sea posible, para maximizar el aprendizaje

### Progresión recomendada

1. **Semana 1-2**: Problemas básicos de rumbo y distancia
2. **Semana 3-4**: Problemas de posición simples
3. **Semana 5-6**: Navegación costera
4. **Semana 7-8**: Problemas combinados
5. **Semana 9+**: Problemas avanzados y astronómicos
